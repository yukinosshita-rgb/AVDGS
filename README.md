# Advanced Visual Docking Guidance System (A-VDGS)

Capstone project for aviation training environments.

## Tech Stack
- ASP.NET Core MVC
- C#
- SQL Server
- Python
- YOLOv8
- OpenCV

## Solution Contents
- Web application
- Business logic layer
- Data access layer
- Models
- Database scripts

## Database Setup
1. Open SQL Server Management Studio
2. Run files in the `Database` folder in order
   - 01_Create_Database.sql
   - 02_Tables_and_Relationships.sql
   - 03_Seed_Data.sql

## Notes
- This is a training-focused academic prototype.
- Not for operational airport deployment.
